# File-Tree
A file tree representation rooted at the file specified by the user

Sample Usage:
```
$ ../print_ftree .
```

Sample Output:
```
===== . (d700) =====
  ftree.c (-640)
  Makefile (-640)
  ===== temp (d750) =====
    hello.txt (-640)
    temp_link (l777)
    ftree_linked.c (-640)
  ftree.h (-644)
  print_ftree.c (-644)
```
